# PlanBuddy – AI-Powered Goal Planner

**PlanBuddy** helps you turn any goal into a structured, actionable plan using AI. Enter your goal, pick a time horizon, and get prioritized tasks with due dates — all saved locally.

---

## Features

- AI-generated task breakdown
- Priority levels (High/Medium/Low)
- Due dates & progress tracking
- Local persistence (works offline)
- Clean, mobile-first UI

---

## Tech Stack

| Layer     | Technology                     |
|---------|-------------------------------|
| Mobile  | React Native + Expo + TypeScript |
| Backend | Node.js + Express + TypeScript |
| Storage | AsyncStorage (mobile)         |

## Usage

To run the frontend 

1.Run npm i or yarn in the terminal for installing the packages
2.Connect your app with simulator/emulator and run npx expo run or npm start
3.The app will start working

For backend

1.Run the npm i in the terminal for installing the packages
2.Run npm run start in ther terminal
3.The server will start running on http://localhost:8787

Now you can generate the plan and test the app
